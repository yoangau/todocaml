# todocaml
My first OCaml project.
A simple todo list application written in OCaml.

## Commands

### Build

```sh
dune build
```
### Watch

```sh
dune build --watch --terminal-persistence=clear-on-rebuild 
```

### Run

```sh
./_build/default/bin/todocaml.exe 
```

```sh
dune exec ./bin/todocaml.exe   
```
